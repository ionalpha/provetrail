# Clients

Language clients and verifiers for the Provetrail standard. Each is published under the canonical bare name `provetrail` in its ecosystem.

- `npm/` -> the `provetrail` package on npm (JavaScript / TypeScript)
- `python/` -> the `provetrail` package on PyPI (Python)

These are currently placeholders (version `0.0.x`) that reserve the canonical names and point to the specification. They become the cross-language verifier once the cryptographic layer of the standard lands. The specification and conformance suite live at the repository root.
